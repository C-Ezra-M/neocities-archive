Hello! It's time to introduce Version 2 of my website!

## Important changes

- The structure of the website is now different - some URLs have changed.
    - For example, `/pokemon-pl` became [`/pl/pokemon`](/pl/pokemon).
    - Note that all of the old links will be redirected. This is done for future-proofing — while I decided on that myself, a stronger argument is that [my translation of <cite>Paweł i Gaweł</cite> into Toki Pona](/tok/jan-pawe-kawe) was linked in [the October 6 issue of <cite>sin kulupu</cite>](https://sin-kulupu.pona.la/2025/q/).
        - Theoretically, I could ask kala Asi to update the link, but redirection is better, because it provides backwards compatibility out of the box, and <cite>sin kulupu</cite> is also distributed by email.
- A [sitemap](/map) was added!
- Added the [about me](/me) page!
- Added the [Gardner theme](/?theme=gardner), meant to resemble typewritten output and named for a lesser-known typewriter manufacturer. It's a work in progress, though, since some details will have to get ironed out.
- Touched up the Tiny Orange theme a bit, changed the font to Arial, and made it the default!
- Disabled SmartyPants, so all quotes will be simple quotes instead of typographical ones.
- Modified the *nasin nanpa* font, removing glyphs for arrow characters that were empty instead of absent. An unofficial fix to [issue 61](https://github.com/etbcor/nasin-nanpa/issues/61).
