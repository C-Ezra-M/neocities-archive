## What is MUD?
<dfn>MUD</dfn> (short for <dfn>Mutually Unhinged Destruction</dfn>) is a camp started in 2025, jointly hosted by C.Ezra.M, TGiovans, and INEI.
> *C.Ezra.M triumphantly re-enters the camp hosting scene with Mutually Unhinged Destruction, MUD for short! It is going to be a fun experience — for both you (the 40 contestants) and the host, who is exploring ways to improve his camps! One thing this camp has going for it is the 12 token types — pretty diverse, but not too chaotic!*

### Extra information on token usage

When multiple tokens are used in an elimination, they apply in the following order:
1. Kamikaze Token
2. Revenge Token
3. Win Token
4. Split Token
4. Guess Token
5. Swap Token
6. Immunity Token

During a challenge, multiple tokens may be used. But in an elimination, only one token may be used.
